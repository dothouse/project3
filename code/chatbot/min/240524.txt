1. 감성분류 연습 코드로 네이버 질의응답 데이터를 가지고 다시 의도 분류 연습
train data(label)이 있는 데이터가 없어서 '질문' 컬럼을 가지고

음식/재료 => 0
추천 => 1
기타 => 2
로 의도 분류 라벨 생성하여 완성된 데이터로 예측 결과값 확인

2. [다음주 목표] 의도분류된 것 중에 0으로 분류된 질문을 음식과 재료로 세분화
 1) 음식명과 재료명을 단어사전으로 구축
 2) 1(추천)로 분류된 질문을 세분화
  - [요리방법별명] 컬럼 기타 -> 튀김
  - [요리종류별명] 컬럼에 '분식' 카테고리 추가 => 떡볶이, 떡꼬치, (위에서 바꾼)튀김, 김밥, 라면, 라볶이, 만두, 쫄면
  - 요리종류별명 통일 ex) 약과,식혜 => 디저트
  - [요리종류별명] 빵 -> 피자
  - 단어사전은 재료명과 음식명이 겹치지 않게 만들기
